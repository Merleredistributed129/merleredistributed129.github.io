## ❓ Frequently asked questions

**Why does the page take time to load?**
The tool downloads a large file containing the character data when you open it for the first time. This file is approximately 53 megabytes. Once downloaded, your browser stores this data in its temporary memory to speed up future sessions.

**Is my computer safe?**
Yes. The software runs entirely on your local machine. It does not send your identity settings or character data to an external server. The processing happens within the browser environment.

**How do I update the software?**
To get the latest version, visit the download link provided above. Download the new version, extract the files to a new folder, and run the index.html file as you did before. You can delete the old folder if you no longer need the previous version.

**The model looks distorted, what is wrong?**
This usually occurs if your graphics card does not support WebGL or if your browser hardware acceleration is off. Check your browser settings menu to ensure hardware acceleration is active.

**Can I save my work?**
The current version focuses on viewing and editing in real-time. To save your progress, take a screenshot of your browser window. Future updates will introduce standard export file formats for 3D modeling programs.
